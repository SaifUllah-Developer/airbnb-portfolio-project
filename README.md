# Airbnb Portfolio Project

This is a **portfolio project** simulating an Airbnb-style web application with **Host** and **Guest** roles.  
It is built using **Node.js, Express.js, MongoDB, EJS, and Tailwind CSS**.  

> ⚠️ Note: This is a learning/portfolio project and not a full production-ready MERN stack app.

---

## Features

### Guest
- View homes with details: name, location, price, rating, image, description  
- Add homes to favorites (one home can be added only once)  
- View and remove favorites  
- Can see only logged-in pages

### Host
- Add, edit, delete own homes  
- View only their own home list  
- Secure login and session management  

---

## Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Frontend:** EJS, Tailwind CSS  
- **File uploads:** Multer  
- **Authentication:** Session-based login
